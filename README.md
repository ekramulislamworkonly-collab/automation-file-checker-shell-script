# File Checker Automation Script

A simple Bash automation script that checks whether a file exists in the current directory.  
If the file is not found, the script automatically creates it. This project demonstrates the use of Bash scripting, conditional statements, file handling, and automation concepts in Linux.

---

## 🚀 Features

- Checks whether a file already exists
- Automatically creates the file if missing
- Uses conditional (`if`) statements
- Beginner-friendly automation project
- Reduces manual file creation tasks
- Lightweight and fast execution

---

## 🛠️ Technologies Used

- Bash Scripting
- Linux Terminal Commands
- Shell Automation
